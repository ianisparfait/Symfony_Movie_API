# Symfony API

## Install project
In your terminal:

1. composer install
2. yarn install
3. yarn run watch

Open new terminal and:  

4. symfony server:start ( need symfony CLI )
/!\ If you does'nt use symfony CLI, replace domain in VUEJS app /!\   

- assets/vue/Views/App/Application.vue => `domain: "127.0.0.1:8000/"`
